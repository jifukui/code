for循环

for循环是JavaScript语言的循环语句使用的是数组的特性进行对数组元素的遍历

let a=[1,2,3,4,5,6,7,8,9];
let value=0;
for(i=0;i<a.length;i++){
value+=a[i];
}
return value;

foreach

foreach有2个参数，第一个参数是一个回调函数可以有三个参数第一个参数是当前值第二个参数是当前的索引第三个参数是数组对象 除了第一个参数之外其他两个参数为可选参数；第二个参数为传递给参数的值可以不写默认值为this，foreach函数中不允许使用 break和continue语句，同时没有返回值。与for循环不同的是这个是数组的方法。

let a=[1,2,3,4,5,6,7,8,9];
var data=0;
a.forEach((val,ind)=>{
data+=val;
},data);

map

map方法其实和foreach方法的功能和参数其实大致一致，但是他有返回值，返回值是由函数每次执行时返回值组成的数组

let a=[1,2,3,4,5,6,7,8,9];
var data=0;
a.map((val,ind)=>{
data+=val;
},data);
